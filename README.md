# 🧩 JSON-LD Schema Collection for AIO & SEO

**Structured data templates to maximize entity recognition and AI Overview visibility.**

As Search Engines evolve into Answer Engines (AIO/SGE), structured data is no longer optional—it is the language we use to speak to the AI. This repository contains a comprehensive list of JSON-LD schemas tuned for entity clarity and Rich Results.

## ⚡ Goals
* **Disambiguation:** Clear definitions of `About`, `Mentions`, and `SameAs` to train the Knowledge Graph.
* **Visibility:** Maximize CTR with Star Ratings, FAQs, and Price snippets.
* **Efficiency:** Copy-paste ready templates for rapid deployment.

## 📂 Included Schemas
* `Organization` (with Knowledge Graph links)
* `Article` / `NewsArticle` (Author vectoring)
* `Product` (Merchant Center aligned)
* `FAQPage` (AIO targeted)
* `BreadcrumbList` (Site structure)

## 🚀 Usage
1.  Select the desired schema template from `/src`.
2.  Replace the `{{variable}}` placeholders with your dynamic data.
3.  Inject into the `<head>` of your document.
